# AgroSense AI Landing Page - Premium Pitch Version

This version was adjusted for a premium, luxury presentation style suitable for a pitching masterclass, customer demos, partner conversations, and early acquisition.

## Included files

- `index.html` - Landing page structure
- `styles.css` - Premium visual styling, responsive layout, mobile optimization
- `script.js` - Mobile menu and demo form feedback

## What to customize before publishing

1. Replace all WhatsApp placeholder URLs using the format:
   `https://wa.me/COUNTRYCODEPHONENUMBER?text=Your%20message`
2. Replace `https://forms.gle/example` with the real survey URL.
3. Confirm the destination email before publishing. It is set to `agrosenseneurallabs@gmail.com`.
4. Connect the waitlist form to a live tool such as Google Forms, Airtable, Tally, Typeform, HubSpot, Supabase, Firebase, or your backend.
5. Publish the folder to Vercel, Netlify, GitHub Pages, Hostinger, Cloudflare Pages, or another static hosting provider.

## Main objective

Create a professional, shareable public landing page that farmers, partners, NGOs, and customers can visit during pitch sessions, demos, and acquisition conversations.


Brand asset included:
- assets/agrosense-logo.png -> official logo image used in the header, footer, and favicon.


## Waitlist email submission

The waitlist form is configured to submit all named fields through FormSubmit to:

```html
https://formsubmit.co/agrosenseneurallabs@gmail.com
```

The first submission may require email confirmation from FormSubmit. The selected pricing plan is included in the email through the `selected_plan` field.
